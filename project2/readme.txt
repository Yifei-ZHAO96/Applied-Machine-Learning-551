There are 5 ipynb files in this zip:
1)'BNB.ipynb ', the Bernoulli Navie Bayes we developed from scratch.
2)'Ensemble.ipynb ', which is uesd to compare performance of different models (model evaluation) and ensemble them.
3)'MNB_coefficient.ipynb ', which concludes the way we used to turning parameters used in multinomial navie bayes.
4)'Test for Kaggle.ipynb ', which generates 2 cvs files used to submit to Kaggle. Note that we only upload 'reddit_test_kaggle', 'test_launch_kaggle' is a temporal file.
5)'preprocessing.ipynb', which includes ways we used to preprocessing features: implementation of sentiment, lemmatization, tf-idf
